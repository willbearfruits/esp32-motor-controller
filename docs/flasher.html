Browser Not Supported

Web Serial API is required for flashing. Please use one of these browsers:

  • Google Chrome (v89+)
  • Microsoft Edge (v89+)
  • Opera (v76+)

Safari and Firefox do not support Web Serial.

Flash Firmware

Install ESP32 Motor Controller firmware directly from your browser. No additional software required!

Your browser doesn't support Web Serial Serial access denied by browser

Requirements

  • Chrome, Edge, or Opera browser
  • USB cable connected to ESP32
  • Appropriate USB drivers installed
  • No other program using the serial port
Note: If prompted, you may need to hold the BOOT button on your ESP32 while clicking "Install" to enter bootloader mode.

After Flashing

  1. Wait for reboot - Device will restart automatically
  2. Find WiFi network - Look for "MotorController" SSID
  3. Connect - Password is motorctrl
  4. Open browser - Go to http://192.168.4.1
  5. Configure motors - Use the web dashboard to set up your motors
Tip: You can change WiFi settings later in the Settings page to connect to your existing network.
